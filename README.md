dotvim
======

Vim configuration files

## Installation

    git clone git:/github.com/sugaryourcoffee/dotvim.git ~/.vim

## Create symlink

    ln -s ~/.vim/vimrc ~/.vimrc

## Get submodule

    cd ~/.vim
    git submodule init
    git submodule udpate
